# Team Inbox

Drop files here for the AI team to process.

AXIOM checks this folder at the start of every session,
reports what's here, and asks Ron what to do with it.

## What to drop here
- Documents to analyze or summarize
- Images to review
- URLs to research
- Task briefs (.md files)
- Hire requests (use format in Team/hiring/process.md)
- Any file you want the team to act on

For consequential work, use `Team/TASK_BRIEF_TEMPLATE.md`.
That template should be the default for strategy, build, audit, legal, and multi-step work.

## Hire Request Format
```
HIRE REQUEST
Role needed: [what this agent does]
Trigger: [when should AXIOM route to this agent]
Key outputs: [what this agent delivers]
Named: [name if you have one]
```
