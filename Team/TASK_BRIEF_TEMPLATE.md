# Task Brief Template

Use this template for consequential work placed in `Team Inbox/`.
If the task is strategic, technical, legal, or likely to create a deliverable,
use the full template.

## Required
- **Task Name**:
- **Objective**:
- **Context**:
- **Why This Matters Now**:
- **Desired Deliverable**:
- **Definition of Done**:
- **Constraints**:
- **Deadline / Urgency**:

## Optional But Helpful
- **Relevant Files / Links**:
- **Known Risks**:
- **What Good Looks Like**:
- **What To Avoid**:
- **Preferred Agent Or Route**:

## Example
```md
Task Name: ProfilePays MVP billing decision
Objective: Decide whether MVP should launch freemium or paid-first
Context: Early-stage marketplace, no live users yet
Why This Matters Now: Billing model affects MVP scope and activation flow
Desired Deliverable: Recommendation memo with ranked options
Definition of Done: One clear recommendation, rationale, risks, and next actions
Constraints: No vague "it depends" output
Deadline / Urgency: Before MVP spec is finalized
```
