What Makes This Novel

No central controller
Each agent runs a fully independent reasoning loop. Coordination is emergent — not orchestrated.
Hardware-heterogeneous
x86 Windows + ARM GB10 + cloud APIs — unified via one file protocol.
File protocol = universal interface
Any agent on any OS with filesystem access can participate. Zero API contracts needed.
Shared dir = 3 roles at once
It is the Environment agents act on, the Memory they recall from, and the Audit Log simultaneously.
One agent's ACT = another's PERCEPTION
Output files become input for all other loops on next poll cycle — no explicit routing.
170K+ messages = training corpus
The audit log is also a multi-agent dialogue dataset, patent evidence, and replayable test suite.

Agents independently converge.
